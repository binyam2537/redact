redact

Protect your prompts.

A local-first browser extension that removes sensitive information, API keys, and PIIs from your chatbot messages before you hit send.

Download .crx View Source
Installation Guide
  • 1
    Download the redact.crx file using the button on the left.
  • 2
    Open a new tab and navigate to chrome://extensions in your browser URL bar.
  • 3
    Toggle on Developer mode in the top right corner of the extensions page.
  • 4
    Drag and drop the downloaded .crx file anywhere onto the extensions page to install it.
